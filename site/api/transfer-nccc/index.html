    NCCCEvaluator¶

    Nearest Class Center Classifier (NCCC) evaluator.

    Computes class centers from support representations and evaluates classification by nearest center in Euclidean space.

    Parameters:

    Name Type Description Default
    device str or device

    Device used for computation.

    		 "cpu"

    compute_class_centers¶

    Compute class centers from full- or few-shot support data.

    Examples:

    import torch
from repx.transfer import NCCCEvaluator

features = torch.randn(100, 64)
labels = torch.randint(0, 10, (100,))

nccc = NCCCEvaluator(device="cpu")
centers_list, selected = nccc.compute_class_centers(
    features,
    labels,
    n_shots=5,
    repeat=3,
)

    Parameters:

    Name Type Description Default
    features (Tensor, shape(num_samples, feature_dim))

    Support feature matrix.

    		 required
    labels (Tensor, shape(num_samples))

    Support labels.

    		 required
    n_shots int

    Number of support samples per class. If omitted, use all support samples in the selected subset.

    		 None
    repeat int

    Number of repeated center computations when n_shots is set.

    		 1
    selected_classes sequence[int]

    If provided, restrict center computation to this class subset.

    		 None

    Returns:

    Type Description
    tuple[list[Tensor], list[int]]

    (centers_per_repeat, resolved_classes) where each centers tensor has shape (num_selected_classes, feature_dim).

    evaluate¶

    Evaluate accuracy using nearest class centers.

    Centers are computed internally by :meth:compute_class_centers from the same input features and labels.

    Examples:

    import torch
from repx.transfer import NCCCEvaluator

features = torch.randn(100, 64)
labels = torch.randint(0, 10, (100,))

nccc = NCCCEvaluator(device="cpu")
accs = nccc.evaluate(features, labels, n_shots=5, repeat=3)
print(accs)

    Parameters:

    Name Type Description Default
    features (Tensor, shape(num_samples, feature_dim))

    Query feature matrix.

    		 required
    labels (Tensor, shape(num_samples))

    Query labels.

    		 required
    n_shots int

    Number of support samples per class used to compute centers.

    		 None
    repeat int

    Number of repeated evaluations when n_shots is set.

    		 1
    selected_classes sequence[int]

    If provided, evaluate only this class subset.

    		 None

    Returns:

    Type Description
    list[float]

    Accuracy for each repeat.
