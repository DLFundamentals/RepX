    RSA¶

    Representational Similarity Analysis in PyTorch.

    Compares two representation spaces by correlating their Representational Dissimilarity Matrices (RDMs).

    Parameters:

    Name Type Description Default
    rdm_metric ('correlation', 'cosine', 'euclidean', 'cityblock')

    Pairwise distance metric used when building RDMs. Options: "correlation" (1 − Pearson r), "cosine" (1 − cosine similarity), "euclidean" (L2 distance), and "cityblock" (L1/Manhattan distance).

    		 "correlation"
    compare ('spearman', 'pearson')

    Correlation method used when comparing RDM vectors.

    		 "spearman"
    Notes
    • Identical representations produce scores near 1.
    • Independent representations usually produce scores near 0.

    compute_rdm(X) ¶

    Compute the Representational Dissimilarity Matrix (RDM).

    Parameters:

    Name Type Description Default
    X (Tensor, shape(n_samples, n_features))

    One row per sample.

    		 required

    Returns:

    Name Type Description
    rdm (Tensor, shape(n_samples, n_samples))

    Symmetric matrix with zero diagonal.

    Examples:

    import torch
from letorch.alignment import RSA

X = torch.randn(5, 3)
rdm = RSA(rdm_metric="correlation").compute_rdm(X)
print(rdm.shape)  # torch.Size([5, 5])

    rsa(X, Y) ¶

    Compute Representational Similarity Analysis (RSA).

    Builds an RDM for each representation matrix, vectorises the upper triangles, and correlates them using the settings from __init__.

    Parameters:

    Name Type Description Default
    X Tensor, shape (n_samples, n_features_*)

    Row-per-sample matrices. Both must have the same number of rows; feature dimensionalities may differ.

    		 required
    Y Tensor, shape (n_samples, n_features_*)

    Row-per-sample matrices. Both must have the same number of rows; feature dimensionalities may differ.

    		 required

    Returns:

    Name Type Description
    r scalar Tensor

    Correlation coefficient in [−1, +1]. Call .item() to get a Python float.

    Raises:

    Type Description
    ValueError

    If X and Y have different number of samples.

    Examples:

    import torch
from letorch.alignment import RSA

rsa = RSA(rdm_metric="correlation", compare="spearman")
X = torch.randn(20, 64)
Y = torch.randn(20, 128)

score = rsa.rsa(X, Y)
print(score.shape)  # torch.Size([])

    CKA¶

    Centered Kernel Alignment in PyTorch.

    Compares two representation spaces with centered kernel alignment using the debiased HSIC estimator.

    Parameters:

    Name Type Description Default
    kernel str

    Kernel used to build the similarity matrix.

    "linear" (default) K = X @ Xᵀ. Invariant to orthogonal transformations and isotropic scaling of the rows.

    		 'linear'
    Notes
    • Invariant to orthogonal transforms and isotropic scaling with linear kernel.
    • Requires at least 4 stimuli (rows).

    cka(X, Y) ¶

    Compute Centered Kernel Alignment (CKA).

    Builds a kernel matrix for each representation, zeros the diagonals, and computes the normalised debiased HSIC.

    Requires at least 4 stimuli (rows) due to the n*(n-3) denominator in the unbiased HSIC estimator.

    Parameters:

    Name Type Description Default
    X Tensor, shape (n_samples, n_features_*)

    Row-per-sample matrices. Both must have the same number of rows; feature dimensionalities may differ.

    		 required
    Y Tensor, shape (n_samples, n_features_*)

    Row-per-sample matrices. Both must have the same number of rows; feature dimensionalities may differ.

    		 required

    Returns:

    Name Type Description
    score scalar Tensor

    CKA score. Returns 1.0 for identical representations, ≈ 0.0 for independent ones. Call .item() for a Python float.

    Raises:

    Type Description
    ValueError

    If X and Y have different number of samples or n < 4.

    Examples:

    import torch
from letorch.alignment import CKA

cka = CKA(kernel="linear")
X = torch.randn(20, 64)
Y = torch.randn(20, 128)

score = cka.cka(X, Y)
print(score.shape)  # torch.Size([])
