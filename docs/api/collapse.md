# Collapse

Collapse-based representation metrics.

Collapse metrics characterize representation geometry and class-structure
behavior.

- **CDNV** measures class variance normalized by inter-class distance.
- **Directional CDNV** measures variance along class-to-class directions.
- **ETF Deviation** measures distance from simplex ETF geometry.
